# 🤖 AI Voice Task Manager

## 📌 Overview

AI Voice Task Manager is a desktop application built using Python and PyQt5 that helps users manage tasks using voice commands. It integrates Groq LLM for natural language understanding, allowing users to create, edit, and manage reminders through an interactive voice assistant.

---

## ✨ Features

- 🎙️ Voice command support
- 🤖 AI-powered task management using Groq LLM
- 📅 Create and manage reminders
- 🔔 Real-time reminder notifications
- 💬 Text-to-Speech responses
- 🖥️ Modern PyQt5 desktop interface

---

## 🛠️ Technologies Used

- Python
- PyQt5
- Groq API
- SpeechRecognition
- pyttsx3
- JSON

---

## 🚀 Installation

```bash
git clone https://github.com/smitnimje/AI-TandR.git

cd AI-TandR

pip install -r requirements.txt

python agent_frontend.py
```
